Настольные источники питания

Настольные источники питания EDU36311
Источник питания постоянного тока 90 Вт

Краткая информация

EDU36311A компании Keysight – это источник питания с тремя выходами, который является частью семейства Keysight Smart Bench, объединяющего четыре уникальных прибора с одним мощным графическим интерфейсом для подключения, управления и захвата результатов измерений. Компактная конструкция источника питания позволяет разместить его в любом месте на рабочем столе. Большой 7-дюймовый дисплей позволяет видеть выходные параметры под любым углом. Все выходы независимые. Их можно соединять последовательно для получение большего напряжения, или параллельно для получения большего тока, или использовать по отдельности для получения напряжения от 0 до ±30 В. Функция коммутации выходов в заданной последовательности позволяет включать и выключать их в указанном порядке. Например, можно сначала подать питание на схему управления, а затем на остальную часть вашего устройства.

• ПО PathWave BenchVue Power Supply для дистанционного управления и регистрации данных
• уровень шума и пульсаций < 350 мкВср.кв. для малошумящего выхода
• точное регулирование тока и напряжения с отличными возможностями программирования и считывания показаний
• уверенность в результатах измерений за счет превосходного коэффициента стабилизации
• сохранение файла настроек на USB накопителе для воспроизводимости измерений
• простое соединение через порты USB и LAN (LXI Core)

Основные характеристики:
• Количество каналов: 3 независимых
• Общая мощность: 90 Вт
• Пульсации и шум: < 1 мВ СКЗ
• Точность: 0,05% + 1,8 мВ
